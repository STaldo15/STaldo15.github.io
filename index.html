Opciones del Editor

Archivo

Nuevo

Abre un nuevo campo de edicion de texto, se puede acceder con el atajo CTRL+N.

Abrir

Abre el explorador de archivos que permitira elegir un archivo de texto para abrirlo en el editor de texto, se puede acceder con el comando CTRL+O.

Guardar

Guarda los cambios en el archivo actual, sino hay un archivo existente con ese nombre, abrira el explorador de archivos para elegir su ruta de guardado, se puede acceder con el comando CTRL+S.

Guardar como

Abre el explorador de archivos que permitira elegir la ruta de guardado del archivo actual.

Imprimir

Abrira una ventana donde se puede seleccionar la especificaciones para imprimir el archivo actual, se puede acceder con el atajo CTRL+P.

Salir

Una vez comprobado que el archivo actual esta guardado, cerrara el editor, se puede acceder con el atajo CTRL+Q.

Editar

Deshacer

Revierte la ultima modificacion hecha en el archivo actual, se puede acceder con el comando CTRL+Z.

Rehacer

Restaura la accion hecha por la opcion Deshacer, se puede acceder con el comando CTRL+Y.

Cortar

Corta el texto seleccionado, se puede acceder con el comando CTRL+X.

Copiar

Copia el texto seleccionado, se puede acceder con el comando CTRL+X.

Pegar

Pega el texto copiado o cortado anteriormente, se puede acceder con el atajo CTRL+V.

Buscar

Abre una ventana donde se puede escribir una palabra que desee buscar y al seleccionar "Aceptar" sera llevado hasta los puntos del texto donde concuerde la palabra especificada, se puede acceder con el atajo CTRL+F.

Ir a Linea...

Se abre una ventana donde se escribe la linea del texto a la que desea ir rapidamente, se puede acceder con el atajo CTRL+T.

Seleccionar Todo

Selecciona todo el texto actual, se puede acceder con el atajo CTRL+E.

Run

Run Project

Verifica si el archivo actual esta guardado, posteriormente comienza la ejecucion del analizador Lexico y Sintactico del editor sobre el contenido del area de texto.

Acerca de la Sintaxis del Editor

Palabras Reservadas

Kami

Esta sera la palabra reservada con la que iniciara el codigo, seguido de {} y dentro de ellas escribiremos todo el nuestro codigo, Se puede comparar con el main de C++, ejemplo: Kami{ Codigo }.

Tenshi

Esta palabra clave tendra la misma funcion del condicional IF de otros lenguajes como por ejemplo C++, tiene la misma sintaxis solo sustituyendo IF por Tenshi, ejemplo: Tenshi(condicion){sentencia}.

Akuma

Palabra clave con la misma funcion que la sentencia ELSE, en caso de no cumplirse la condicion de Tenshi, realizara la sentencia Akuma en caso de tenerla, ejemplo: Tenshi(condicion){sentencia} Akuma{sentencia}.

Yomu

Palabra clave con la funcion de imprimir, su sintaxis funciona muy parecido al cout de c++, ejemplo: Yomu<<'hola mundo';.

Kaku

Esta palabra reservada sera la guarde los datos introducidos por el usuario, la sintaxis sera igual a cin del lenguaje C++, ejemplo: Kaku>>variable;.

Sora

Palabra clave que tiene la funcion de un ciclo While de muchos lenguajes como por ejemplo C++, ejemplo: Sora(condicion){Sentencia}.

Neko

Esta palabra se usa para definir una variable de tipo Entero, su declaracion es similar al lenguaje C, ejemplo: Neko variable;.

Inu

Esta palabra se usa para definir una variable de tipo Flotante, su declaracion es similar al lenguaje C, ejemplo: Inu variable;.

Sintaxis especificas

Comentario unilinea

Para hacer declarar un comentario de 1 se tiene que empezar con un # y concluira al hacer salto de linea, ejemplo: #comentario unilinea.

Comentario Multilinea

Para especificar este comentario se comienza con ** y se termina con de igual con **, ejemplo: **Comentario Multilinea**.

Punto y Coma

El punto y coma; es el signo de fin de sentencia. Cualquier expresión legal termina por un punto y coma es interpretado como una sentencia, conocidas como sentencia-expresión. La expresión se evalúa y el resultado se descarta; si no tiene efectos colaterales. Ejemplo: a + b; // evalúa a + b, descarta el resultado.

Igual que

Para comparar si una variable es igual a otra se usa ==, ejemplo: Sora(i==1){sentencia}.

Mayor

Para comparar si una variable es mayor a otra se usa >, ejemplo: Sora(i>1){sentencia}.

Menor

Para comparar si una variable es menor a otra se usa <, ejemplo: Sora(i<1){sentencia}.

Mayor igual

Para comparar si una variable es mayor o igual a otra se usa >=, ejemplo: Sora(i>=1){sentencia}.

Menor igual

Para comparar si una variable es menor o igual a otra se usa <=, ejemplo: Sora(i<=1){sentencia}.
